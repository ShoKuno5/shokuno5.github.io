## What is collected

This site uses privacy-first analytics to understand broad reading patterns.

- Page views
- Scroll milestones (50%, 90%)
- Citation panel opens
- Comment loader clicks
- Related-post link clicks

## What is not collected

- No cookies
- No user accounts
- No fingerprinting
- No cross-site tracking IDs

## Data handling

Events are aggregated by day and page path. The analytics endpoint stores only coarse event counts.
